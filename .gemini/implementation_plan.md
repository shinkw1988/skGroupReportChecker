# グループ報告チェッカー - 実装計画

## プロジェクト概要
スマートフォン向けに最適化されたグループ（班）の集合状況チェック用Webアプリ

## 技術スタック
- **フレームワーク**: Vite + Vanilla JavaScript
- **スタイリング**: Vanilla CSS（水色ベース、モバイルファースト）
- **データ永続化**: LocalStorage
- **ターゲット**: スマートフォン（縦長画面）

## ファイル構成
```
/
├── index.html          # メインHTML
├── style.css           # スタイルシート
├── app.js             # アプリケーションロジック
├── background.jpg     # 背景画像（提供された画像を使用）
└── package.json       # プロジェクト設定
```

### ステップ5: 機能詳細

#### チェック画面
1. **ヘッダー**
   - タイトル: "グループ報告チェッカー"
   - 選択解除ボタン: 全チェックをクリア

2. **リスト行（各班）**
   - 順番表示エリア（チェック順の数字）
   - 班名（"1班" または "A班"）
   - チェックボタン
   - 備考欄（プレースホルダー: "人数など"）

3. **フッター**
   - 設定ボタン

#### 設定画面
1. **班タイプ選択**
   - ラジオボタン: "1班タイプ" / "A班タイプ"

2. **班の数設定**
   - 数値入力（10キータイプ）
   - 範囲: 1-30
   - バリデーション: 31以上で警告表示

3. **初期設定**
   - デフォルト: "1班タイプ"、12班
