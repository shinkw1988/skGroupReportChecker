# skグループ報告チェッカー (skGroupReportChecker)

グループ（班）の集合状況や報告状況を、スマートフォンで迅速にチェック・管理するためのWebアプリケーションです。
現場での使いやすさを重視し、直感的な操作と視覚的なわかりやすさを追求しています。

---

**このアプリは、Google DeepMindの次世代AIコーディングアシスタント「Antigravity」との共同作業によって開発されました。**

## 💡 開発の背景 / 作者について
作者は**東京都内の小学校教員**です。引率時や校外学習、学校行事において、多数の班の集合報告を正確かつスマートに記録したいという現場のニーズから、このアプリを着想・作成しました。
プログラミングの専門知識をAI（Antigravity）がサポートすることで、現場に即したツールを短期間で形にすることを目指しています。

## 🚀 特徴

- **かんたんチェック**: タップするだけで班の報告状況を記録。
- **報告順の自動記録**: チェックした順番が自動的に番号で表示されます。
- **備考メモ機能**: 各班ごとに自由にテキストを入力可能。
- **柔軟な設定**:
  - 班のタイプ選択（「1班」タイプ または 「A班」タイプ）
  - 班の数設定（最大30班まで対応）
- **オフライン対応 (PWA)**: サービスワーカーにより、電波の不安定な場所でも動作します。
- **自動保存**: 入力内容や設定はブラウザに自動保存されるため、タブを閉じても安心です。
- **スタイリッシュなデザイン**: 水色を基調とした清潔感のあるデザインと、透過エフェクトを活用したUI。

## 🛠 使用技術

- **Core**: HTML5, Vanilla CSS, JavaScript (ES modules)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **PWA**: Service Workers / Web App Manifest
- **Design**: Google Fonts (Yusei Magic)

## 📦 セットアップ・開発

このプロジェクトは `Vite` を使用して構築されています。

### 1. 依存関係のインストール

```bash
npm install
```

### 2. ローカル開発サーバーの起動

```bash
npm run dev
```

### 3. ビルド（本番用）

```bash
npm run build
```

## 📱 使い方

1. アプリを開くと班のリストが表示されます。
2. 報告があった班の右側にあるチェックボタンをタップします。
3. チェックした順番に番号が振られます。
4. 「選択解除」ボタンで全てのチェックをリセットできます。
5. 「設定」画面から、班のタイプや数を自由に変更可能です。
